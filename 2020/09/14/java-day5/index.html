面向对象OOP（Object-oriented programming）

面向对象的三大原则——封装、继承、多态

1、封装——类私有化——private

  • 构造方法—抽象类是有构造方法的，在实现他的子类创建对象的时候运行常用与给成员变量赋值

  • this指定当前对象的引用地址，构造方法之间的相互引用

2、继承——子类继承父类功能，并扩展自己的功能
3、多态——子类重写了父类的方法，同一个父类不同的实现类。
